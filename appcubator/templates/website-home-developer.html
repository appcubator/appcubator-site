Don't change your workflow, just make it more efficient.

Appcubator is not a new MVC framework or a language. It's a tool that will help you build things more quickly. Just drag and drop the UI elements, built-in authentication forms, forms that are already bound to the back-end and query lists. Appcubator will generate the code for you.

Some of the functionality includes:

  • Define database tables
  • Drag and drop front-end editor
  • Easily add JS/CSS/HTML pieces to your page
  • Built in User Authentication and Admin panel
  • Generates downloadable Django code
  • Easy deployment and hosting
