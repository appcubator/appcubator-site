    We broke down StudyCloud app into pieces and made screencasts explaining each piece.

    • Part 1: Intro and Tables

      In this walkthrough, we show a user logging into our interface and setting up their tables for a test sharing website. This video will introduce you to Appcubator and show you how to set up your database.
      Start Now »

    • Part 2: Themes and Pages

      In this walkthrough, we show how users set up themes and create the pages that they will need for the web application.

      This video will introduce you to: themes, page access, editing your URLs, adding Page IDs to create user/table specific pages (similar to different Facebook profiles for different users), creating new pages.

      Start Now »

    • Part 3: Editor and Design Elements

      In this walkthrough, we show how to use our editor to design a homepage with our drag and drop interface.

      This video will show you how to: Use our Drag and Drop Editor, Upload images, Modify Headers and Text Styling, Create and Edit your Navigation Bar

      Start Now »

    • Part 4: Add Authentication

      In this walkthrough, we show how to use our editor to set up your login and sign up forms with our drag and drop interface.

      This video will show you how to: Use our Drag and Drop Editor, Drop Login and Sign Up Forms, Edit Login and Sign Up Forms (where they direct), Edit your Footer, Test Run your Application

      Start Now »

    • Part 5: Search and Lists

      In this walkthrough, we show how to use our editor to design a search page where users can search and display information in your database.

      This video will show you how to: Use our Drag and Drop Editor, Use Search Forms, Use Search Result Lists, Use the Table {{Tags}} to change how your information is displayed

      Start Now »

    • Part 6: Create Forms and Uploads

      In this walkthrough, we show how to use our editor to design a create form page.

      This video will show you how to:Use our Drag and Drop Editor, Create a Link to an internal page, Use a Create Form for users to input information into the database, Modify your create form's layout using the example of an upload test form.

      Start Now »

    • Part 7: Profile Page and Page Context Data

      In this walkthrough, we show how to use our editor to design a test 'profile page' and how to add page context data. This is data that is specific for one particular tweet, test, or user. Check your URL to make sure you have added your page context id shown in a {{tag}}

      This video will show you how to: Add your Page Context Data to create a table-based 'Profile Page'
