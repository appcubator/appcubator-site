How to Use the Theme Editor

Theme editor gives you full control over your interface. It requires knowledge of CSS. It uses LESS to compile the styles on the generated app, but it's not a required skill.

You can access the theme editor on the top right menu of your application. You'll be directed to the Theme Editor when you click the "Theme Editor" button.

Base CSS

Base CSS is the CSS that will directly be injected to your page. It should include the overall styling of your page, such as the styling of the body tag, navigation bar or footer.

How to style body tag?

To style the <body> tag, it's just enough to add lines on top of the basecss, without any brackets. The lines you write here will automatically be wrapped with body { /* Your CSS */ }.

How to style Navigation Bar?

Navbar on the generated app has a constant semantic HTML, but you are free to style it however you would like.

The HTML for the navigation bar looks like the following:


        <div class="navbar" id="navbar">
        <div class="navbar-inner">
        <a class="brand" href="/" id="brand-name">Design This For Me? </a>
        <ul class="nav" id="items">
        <li> <a href="#">Item 1 </a> </li>
        <li> <a href="#">Item 2 </a> </li>
        <li class="logout-item pull-right"> <a id="logout" href="#">Logout </a> </li>
        </ul>
        </div>
        </div>

The classs and ids here are always constant, so you can style each part of the navbar through those selectors. The template theme will have a simple styling that you can just fill in.

How to style Footer?


        <div class="footer">
        <div class="container">
        <ul class="footer-links">
        <li><a href="#" class="menu-item">Footer Item 1</a></li>
        <li><a href="#" class="menu-item">Footer Item 2</a></li>
        </ul>
        <p class="footer-text muted">Powered by Appcubator</p>
        </div>
        </div>

The classs and ids here are always constant, so you can style each part of the footer through those selectors. The template theme will have a simple styling that you can just fill in.

Adding UI Element Styles

The types of UI elements you can create on the editor are already defined and you can create new classes, or "styling"s, for these UI element types. The stylings you create will be available on the editor, when you click the "Edit Style" button on the widget editor.

How to add images and icons

Wouldn't it be awesome if you put some nice icons to your theme or provide some more statics? It's easy. You can go to the statics section by clicking the "Statics" on the left menu and you will see the interface where you can upload new statics as well as see your already uploaded statics.

Press Cmd+C or Ctrl+C when you see the prompt below, and you can use it anywhere in your CSS by pasting it. It'll be available everywhere.
