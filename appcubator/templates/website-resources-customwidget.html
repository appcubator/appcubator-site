    What can you do with Custom Widgets?

    Custom Widgets let's you everything that can be achieved with JS, CSS and HTML. There is no limit to it.

    How to add Custom Widgets?

    Custom widgets let you freely add custom front-end code to your pages. It is on the gallery, under the Design Elements section

    When it's dropped on the page, the Custom Widget Editor will show up. Alternatively, you can select it on the editor and click on "Edit Custom Widget" button.

    On the custom widget editor, there are three sections: JS, HTML and CSS. Each sections contains a code editor that should be used for adding the corresponding code. If there is an error in the code, the editor will not run it or render the HTML.

    Adding JS

    You can write your Javascript in the corresponding text editor and it'll be added to your page automatically. Some points to bear in mind while writing Javascript on Appcubator:

    • Two most popular libraries, jQuery and Underscore are already included in the page. So you don't need to include them again and bloat your code. You can just start using $ or _ in your code.
    • JS for each Custom Widget is executed in it's own scope. This means that you can not write two different custom widgets JS interfacing with each other. On the other hand, you are free to interface with DOM elements
    Adding HTML

    You can write you HTML in the corresponding text editor and it'll be rendered on your page when you close the editor. The custom HTML is rendered inside a div with widht and height defined by Appcubator's grid system. So if you just want to utilize editor's resizing/moving abilities, you should style your outer element with width:100% and height:100%

    Adding CSS

    You can easily add CSS to style your existing elements or custom HTML you just wrote using Custom Widget. The only limitation right now is that the "position:fixed;" will not work accuratly. We're working on getting around that limitation
