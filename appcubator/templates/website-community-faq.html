Can I make a mobile app with Appcubator? In the future! We are working on a mobile page editor which we'll make available to you as soon as possible.
I don't have any technical experience, but want to learn. Should I still use Appcubator? Yes. Appcubator is a great tool to learn web development concepts. In fact, at least half of our users are people who are new to web development. Just dive into our resources.
I already know how to code, why should I use Appcubator? Making web applications is hard. Even if you are an experienced developer, you might prefer to work on the front-end, back-end or server setup. Appcubator lets you focus on what you enjoy.
How customizable are Appcubator apps? Fully customizable. Appcubator is not a new language or framework. It generates web application code that you'd normally write. You can download the code, and add your own custom CSS, Javascript or Python code. We've put in extra effort to generate code with good style so you can easily customize it.
How portable is the generated code? Very. We generate code backed by industry standard, popular technologies like Django. You can host your generated app on Amazon AWS, Heroku, Dotcloud and others. See a full list here.
What happens to my application's data on Appcubator? You own it completely. You can browse and download it using the admin panel, and when you delete your app, the data is also deleted.
How can I import data into my app? Your app comes with a data-browser built-in, so you can upload your Excel, CSV, XML, or JSON data.
What is the performance of sites hosted on Appcubator? Good. If your site gets slow, we'll scale the performance up for you up to 1500 users. Want more scalable hosting? Our apps are portable, so you can host anywhere, including the industry giants like Amazon for unlimited performance.
How is Appcubator different from Weebly/Squarespace/Wix or other site builders? Appcubator is not a *site* builder per se. It is an "application" builder that is designed to handle dynamic data and user input.
Do you support other languages/frameworks? In the future. Our current focus is on refining the flow of building applications for you first. Then we'll work on generating PHP or Rails applications as well.
Can I refer my friends/collegues to Appcubator? Yes, we'd appreciate if you did! Just go to the Garage page after you signup and click on "Invite Friends" link. If you sign up 3 friends, we'll send you an Appcubator T-Shirt as a token of our appreciation.
