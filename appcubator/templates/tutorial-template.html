  • Go to Data & Forms Page

    This page contains information about the data you want to store, and forms you would like to display to your users.

  • Choose how you want to authenticate users

    When you check the boxes, corresponding authentication buttons will be available on the editor.

  • Create a Tweet Entity

    You need to identify the type of data you would like to store on your database, and create `entities` accordingly.

  • Add `Content` field

    You need to add the fields belonging to this entity. Fields can have different types such as text, number, image and date. Since tweets just consist of text, we will pick "Text" as the type and make it required.

  • Create a "Create Tweet" form

    This will also show up on your UI editor.

  • Go to Themes Page and Choose One from the Gallery

    We will be adding many more of these soon. We also would like to help you if you already have designs by translating them into private themes.
